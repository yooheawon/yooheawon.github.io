보유 기술

JAVA

Model1 방식의 jsp으로만으로 게시판을 만든 것을 시작으로, Model2 mvc방식으로 기본 자바 servlet으로 개인프로젝트, 팀프로젝트를 만들고 마지막으로 Spring 프레임 워크를 팀프로젝트로 만들면서 web개발에 있어 자바의 역할과 동적 웹페이지 서비스의 구조를 학습하고 있습니다.

DataBase

Oracle DB를 중심으로 학습하였고,

HTML&CSS
